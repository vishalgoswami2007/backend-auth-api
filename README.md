# Backend Authentication API

A secure backend authentication REST API built using Node.js, Express.js, MongoDB, JWT, and bcrypt.

## Features

* User Registration
* User Login
* Password Hashing
* JWT Token Authentication
* MongoDB Database Integration
* Clean MVC Folder Structure

## Tech Stack

* Node.js
* Express.js
* MongoDB
* Mongoose
* JWT
* bcryptjs

## API Endpoints

### Register User

POST /api/auth/register

### Login User

POST /api/auth/login

## Installation

```bash
npm install
npm run dev
```

## Environment Variables

Create `.env` file:

PORT=5000
MONGO_URI=your_mongodb_uri
JWT_SECRET=your_secret_key
